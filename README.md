# Sign-Detection
Sign Detection helps detect whether a person is making a sign(action) or not.
# Dataset
The dataset is created by converting the captured video into frames, with each class(Signed/Unsigned) having 15000 frames.
# Requirements
tensorflow: 2.9.2
numpy: 1.21.6
<br>
streamlit: 1.12.2
<br>
opencv: 4.6.0
